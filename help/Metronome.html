Metronome


superclass: Object


A metronome class which can be controlled from code, or from its built-in GUI. It supports different time signatures, and also exposes a TempoClock object which can be used to synchronize patterns to the metronome.


Creation


*new(num, denom, bpm, out)


Create and return a new Metronome object.


num - the numerator of the time signature.


denom - the denominator of the time signature.


bpm - the number of quarter note beats per minute.


out - the output bus to which audio will be sent.


Usage


play


Start the metronome.


stop


Stop the metronome.


showGUI


display the interactive GUI.


Examples


// Boot the default server.

s = Server.default;

s.boot;


// Create and start a metronome.

m = Metronome.new(4,4,110,0);

m.play;


m.out = 1; // Change the output bus.

m.bpm = 160; // Change the tempo.

m.num = 5; m.denom = 8; // Change the time signature.

m.showGUI; // Load the GUI.


// Sync a Pbind to the metronome.

(

~pattern = Pbind(

\midinote, Pseq([60, 64, 67, 72], inf)

).play(clock:m.clock, quant:4);

)


// Stop the Pbind

~pattern.stop;


// Stop the metronome.

m.stop;
